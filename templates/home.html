Word Hunt Tournament

Multiplayer match mode with separate player screens and a live side-by-side spectator feed.

Player 1 Screen Left station gameplay view Player 2 Screen Right station gameplay view Spectator Screen Dual-board broadcast view

Match Rules

  • Trace words by swiping adjacent letters, including diagonals.
  • Each tile can only be used once per submitted word.
  • Each player has 90 seconds on the same board.
  • Points: 3-4=1, 5=2, 6=3, 7=5, 8+=11.

Meeting Flow

  • Project spectator view on the main display.
  • Players open only their own screen on separate devices.
  • Start a fresh match before each round.
  • Use the winner plus score spread for final ranking.
